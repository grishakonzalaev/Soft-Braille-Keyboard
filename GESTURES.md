# Soft Braille Keyboard — Жесты и команды

Документация по всем жестам брайлевской клавиатуры.

## Раскладка точек (6-точечный брайль)

```
● 1  ● 4
● 2  ● 5
● 3  ● 6
```

- **1, 2, 3** — левый столбец (сверху вниз)
- **4, 5, 6** — правый столбец (сверху вниз)

В 8-точечном режиме добавляются точки **7** (под 3) и **8** (под 6).

---

## Жесты: свайп от точек

*Жест = зажать указанные точки и провести в направлении.*

### Точка 1 (левый верхний угол)

| Жест       | Действие                                      |
|------------|-----------------------------------------------|
| 1 ←        | Курсор влево на символ                        |
| 1 →        | Курсор вправо на символ                       |
| 1 ↑        | Прочитать текущий символ                      |
| 1 ↓        | Сменить тип обратной связи (вибрация/звук)    |
| **Долгое 1 ←** | Уменьшить клавиатуру                      |
| **Долгое 1 →** | Установить/снять метку выделения           |
| **Долгое 1 ↑** | Вкл/выкл авто-заглавные                    |
| **Долгое 1 ↓** | Статистика: строки, слова, символы        |

### Точки 1–2 (левый столбец, верхние две)

| Жест       | Действие                        |
|------------|---------------------------------|
| 1-2 ←      | Курсор влево на слово           |
| 1-2 →      | Курсор вправо на слово          |
| 1-2 ↑      | Прочитать текущее слово         |
| 1-2 ↓      | Сменить режим озвучивания ввода |

### Точки 1–2–3 (весь левый столбец)

| Жест       | Действие                           |
|------------|------------------------------------|
| 1-2-3 ←    | Курсор влево на строку             |
| 1-2-3 →    | Курсор вправо на строку            |
| 1-2-3 ↑    | Прочитать текущую строку           |
| 1-2-3 ↓    | **8-dot:** точка 7 (иначе — неизвестно) |
| **Долгое 1-2-3 ←** | Удалить весь текст (с подтверждением) |
| **Долгое 1-2-3 →** | Переключить литературный/компьютерный брайль |
| **Долгое 1-2-3 ↑** | Голосовой ввод                   |
| **Долгое 1-2-3 ↓** | Сменить брайлевскую таблицу     |

### Точка 4 (правый верхний угол)

| Жест       | Действие                           |
|------------|------------------------------------|
| 4 ←        | Удалить символ (Backspace)         |
| 4 →        | Пробел (двойной — точка и пробел)  |
| 4 ↑        | Режим приватности вкл/выкл         |
| 4 ↓        | **Enter / Go / Send** (новая строка или действие поля) |
| **Долгое 4 ←** | Проверка орфографии влево     |
| **Долгое 4 →** | Проверка орфографии вправо    |
| **Долгое 4 ↑** | Предыдущее предложение по орфографии |
| **Долгое 4 ↓** | Следующее предложение по орфографии |

### Точки 4–5 (правый столбец, верхние две)

| Жест       | Действие                            |
|------------|-------------------------------------|
| 4-5 ←      | Удалить слово                       |
| 4-5 →      | То же, что 4 →                      |
| 4-5 ↑      | Сделать двойной свайп → настройки   |
| 4-5 ↓      | Сделать двойной свайп → переключатель клавиатур |

### Точки 4–5–6 (весь правый столбец)

| Жест       | Действие                            |
|------------|-------------------------------------|
| 4-5-6 ←    | Удалить строку (с подтверждением)   |
| 4-5-6 →    | Сменить режим редактирования        |
| 4-5-6 ↑    | Выполнить текущее действие редактирования |
| 4-5-6 ↓    | **8-dot:** точка 8 (иначе — неизвестно) |
| **Долгое 4-5-6 ←** | Курсор в начало текста        |
| **Долгое 4-5-6 →** | Курсор в конец текста         |
| **Долгое 4-5-6 ↑** | Прочитать весь текст          |
| **Долгое 4-5-6 ↓** | Озвучивание паролей вкл/выкл  |

---

## Режимы редактирования (4-5-6 → и 4-5-6 ↑)

Последовательность режимов при свайпе **4-5-6 →**:

1. **Выделить всё** (Select All)
2. **Копировать** (Copy)
3. **Вырезать** (Cut)
4. **Вставить** (Paste)
5. **Озвучить** (Speak selection)
6. **Удалить** (Delete selection)

Свайп **4-5-6 ↑** выполняет выбранное действие над текущим выделением.

---

## Ввод брайлевских символов

- **Короткое касание** точек 1–6 (без свайпа) — ввод буквы/символа по активной брайлевской таблице.
- В **русском Grade 1**: точка 6 + буква = заглавная; 3456 + паттерн цифры = цифра.

---

## Двойной свайп

Повтор одного и того же жеста дважды за ~1.3 сек считается двойным свайпом и вызывает расширенные действия, например:

- **4-5 ↓** дважды → переключатель методов ввода
- **4-5 ↑** дважды → открыть настройки
- **1-2-3 ↑** дважды → голосовой ввод (если есть разрешение)
- **1-2-3 ←** дважды → удалить весь текст

---

## Особенности

- **Двойной пробел** (4 → дважды): при включённой опции вставляет точку и пробел после буквы или цифры.
- **Озвучивание паролей**: долгое 4-5-6 ↓ переключает озвучивание символов в полях пароля.
- **8-dot режим**: точки 7 и 8 доступны через 1-2-3 ↓ и 4-5-6 ↓.
