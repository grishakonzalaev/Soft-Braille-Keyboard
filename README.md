# Soft Braille Keyboard

Навайбкожено с помощью ИИ!
Метод ввода для Android — экранная брайлевская клавиатура для быстрого и удобного ввода текста.

## Возможности

- **6- и 8-точечный брайль** — литературный и компьютерный режимы
- **Многоязычность** — поддержка нескольких языков (в т.ч. русский Grade 1)
- **Жесты** — навигация, удаление, копирование, вставка, проверка орфографии
- **Голосовой ввод** — ввод текста голосом
- **Go/Send** — отправка в чатах и переход в адресной строке
- **Озвучивание** — вибрация, звук, озвучка введённых символов и слов

## Русская раскладка (Grade 1)

- Точка 6 + буква — заглавная буква
- Цифровой знак (3456) + паттерн — цифры 0–9
